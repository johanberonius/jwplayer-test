Test for mp3 audio source with JW Player 7.8.3 using HLS JS provider

Source:

HLS stream with MP3 audio

Actual behavior

Playbacks freezes about every ten seconds in Firefox 50, seems to work in Firefox 52.

No timeline is visible in Chrome 55 and 57, it says "Live broadcast" even though it's not.

Expected behavior

Audio stream should play just like it does using the Flash provider.

Steps to reproduce

Press play.

JW Player version

7.8.3 with Enterprise key

Stream server

Wowza Streaming Engine 4 Perpetual Edition 4.5.0.01 build18956

Environment

Browsers without native support for HLS but with support for MSE, eg. Firefox 50-52 or Chrome 55-57.
